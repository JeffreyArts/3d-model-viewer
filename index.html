View 3D Model

Use query parameters to control behavior:

  • src: The path to your 3D model file (STL, OBJ). Example: ?src=/models/model.stl
  • color: Set the color of the model (HEX or CSS color name). Example: ?color=red or ?color=FF5733
  • clean: Add this parameter to hide the text and the header. Example: ?clean
